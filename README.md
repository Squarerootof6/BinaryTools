# Require Package
`lmfit,emcee,corner==2.2.3,astropy,tensorflow==2.14.0,numpy<2.3.4,scipy<=1.13.0,pympler`

# Installation

1. install Git LFS
    macOS: brew install git-lfs
    Ubuntu: sudo apt install git-lfs
    Windows: download from https://git-lfs.github.com/
2. pull the repository
    ```
    git clone https://github.com/Squarerootof6/BinaryTools.git
    cd binary_tools
    ```
3. initialize LFS
    `git lfs install`
4. pull lfs
    `git lfs pull`
